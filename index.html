Ejercicio de clase

Vertientes o enfoques más usados para el RWD:

  • Mobile-First: Enfoque que prioriza el diseño para dispositivos móviles.
  • Desktop First: Enfoque tradicional que inicia el diseño en pantallas de escritorio.
  • Content First: Se enfoca en la jerarquía del contenido antes del diseño.

Ejemplos para cada vertiente:

  • Mobile-First: Un sitio de noticias con un diseño limpio para dispositivos móviles.
  • Desktop First: Una plataforma de edición de imágenes en línea.
  • Content First: Un sitio web de recetas que destaca las instrucciones.

Breakpoints para cada vertiente:

  • Mobile-First: 320px, 480px, 768px, 1024px, 1440px.
  • Desktop First: 1440px, 1280px, 1024px, 768px, 480px.
  • Content First: 768px, 1024px, 1280px, 1440px, 1920px.

Conflictos o problemas típicos en el RWD:

  • Dificultad de diseño para diferentes pantallas.
  • Rendimiento lento debido a elementos pesados en dispositivos móviles.
  • Contenido oculto al simplificar el diseño.
  • Navegación complicada al no adaptarla adecuadamente.
  • Necesidad de pruebas exhaustivas en diferentes dispositivos.
